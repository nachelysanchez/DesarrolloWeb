El desarrollo web se refiere al proceso de crear y mantener sitios web y aplicaciones web para su uso en Internet o en una red interna. Los desarrolladores web utilizan una variedad de lenguajes de programación y tecnologías para crear sitios web que sean atractivos, funcionales y fáciles de usar.

Este se divide en Frontend, que es la parte que interactúa con el usuario, tanto imagen como función; y el backend que se refiere a la parte que está en contacto con el servidor.
